AI HTML 可视化修图工具

把 AI 生成的静态 HTML 放进来，像修图一样做最后 10% 的文字、样式和版面微调

这是什么？

这是一个面向 AI 生成 HTML 的可视化精修工具。它作为一段 JavaScript 脚本注入到静态 HTML 页面中，让页面自身具备可视化微调能力。

它不替代 Vue、React 或专业 IDE，只解决 AI 生成页面之后，发布前还差一点点的问题。

使用方式：在任意 HTML 文件底部加入 <script src="editor.js"></script>，然后用浏览器打开即可。

核心功能

AI 文案微调

字号、行高、字间距、字重、对齐方式、文字颜色——滑块和选择器直接调，所见即所得。

背景与间距

背景颜色、不透明度、内外边距——每个属性都有直观的控件，不用写任何 CSS。

尺寸与圆角

宽度、高度、圆角半径——数值输入或滑块调整，实时预览效果。

HTML / PNG 导出

编辑完成后可下载干净 HTML，也可以把当前页或全部页面导出成 PNG 图片。

快捷键

  • Alt + E — 切换编辑模式
  • Alt + T — 编辑选中元素的文字
  • Alt + Z — 撤销上一步修改
  • Alt + Y — 重做上一步撤销
  • Alt + ← / → — 切换上一页 / 下一页
  • Escape — 取消选中 / 退出编辑
  • Alt + C — 复制 HTML 到剪贴板
  • Alt + S — 下载 HTML 文件

适用场景

适合 AI 已经生成了 HTML，但还需要人工过一遍的场景：

  • AI 生成的网页需要手动调整字号、配色和间距
  • 公众号文章排版的最后一步微调
  • PPT 转 HTML 后的布局调整
  • 静态页面的快速原型迭代
