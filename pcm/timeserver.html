🔗 在其他网站获取本页面的时间 JSON 
<iframe id="timeFrame" src="https://你的域名/index.html" style="display:none;"></iframe>
<script>
    const iframe = document.getElementById('timeFrame');
    iframe.onload = () => {
        iframe.contentWindow.postMessage('getTime', '*');
    };
    window.addEventListener('message', (e) => {
        if (e.data && typeof e.data === 'object' && e.data.server_time) {
            console.log('获取到的服务器时间:', e.data);
            // e.data 包含 { server_time, local_time, offset_seconds, timestamp_unix_ms }
            alert('服务器时间: ' + e.data.server_time);
        }
    });
</script>
💡 将代码中的 https://你的域名/index.html 替换为你的实际地址。支持跨域通信，无需后端。
✅ 服务器时间取自 GitHub Pages 响应头 Date · 不依赖第三方 API · 偏移量固定，每秒刷新显示
